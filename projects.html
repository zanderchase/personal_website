Projects

Honors Senior Thesis

Compiled a dataset of millions of tweets by web-scaping Twitter of users throughout South America. Used various visualization and ML techniques to examine lexical differences between regions and built a classifier to predict a person's demographic.

Kelp Finder

Analyzed geospatial data to identify suitable ocean ecosystems for kelp growth on the West Coast of the US. Leveraged NOAA PSL ocean temperature geolocated dataset and US coastal Shapefile data to run an analysis in GeoPandas to identify coastal areas that have had monthly average temperatures between 5–20 degrees over the past five years.

Strava Chatbot

Leveraged Strava API to pull in a user’s run information and create visualizations outlining key performance metrics. Fed user run history into a Langchain LLM agent to create an AI coach chatbot backed by OpenAI models.
